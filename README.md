# movies-app

In this app you will be able to:

1. see the movie description
2. see movie trends
3. search for a movie
4. see movie details 

## Authors

- [@JohnRodriguez](https://github.com/JohnRodriguezM)


## Installation

Install the project dependencies

Using npm
```bash
  npm install
```
Using yarn
```bash
   yarn add
```
